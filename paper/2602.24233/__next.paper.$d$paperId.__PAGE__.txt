2:T2265,<h2>问题动机与背景</h2>
<p>文本到图像（T2I）生成模型在视觉保真度和创造力上取得巨大进步，但在处理包含<strong>复杂空间关系</strong>的长文本提示时仍面临严峻挑战——多对象间的相对位置（左右、前后、上下等）往往需要多次采样才能获得令人满意的结果。在线强化学习（RL）是提升空间理解的有前景方向，但其核心瓶颈在于<strong>缺乏可靠的奖励模型</strong>：</p>
<ul>
<li><strong>人类偏好奖励模型</strong>（HPSv2、PickScore、ImageReward 等）侧重美学与整体文图对齐，无法准确评估多对象空间关系——甚至对空间错误的图像给出更高分数；</li>
<li><strong>VQA 式文图对齐模型</strong>（VQAScore、TIFA 等）同样在复杂空间推理上表现不足；</li>
<li><strong>专有 VLM API</strong>（GPT-5、Gemini-2.5 Pro）准确度高，但每次查询成本过高，无法支撑在线 RL 的高频奖励查询；</li>
<li><strong>开源 VLM</strong>（Qwen2.5-VL-72B 等）存在严重幻觉，在多对象空间推理上不可靠；</li>
<li><strong>规则基奖励</strong>（GenEval）依赖目标检测器，仅支持"A <方位> B"的简单模板，无法泛化至长文本多对象场景，且对遮挡等视觉挑战极度敏感。</li>
</ul>
<p>本文的核心论点：增强图像生成的空间理解，关键在于构建<strong>专用的、可靠的空间奖励模型</strong>。</p>
<h2>SpatialReward-Dataset 数据集</h2>
<p>作者构建了包含 <strong>80K 对抗性偏好对</strong>的 SpatialReward-Dataset，核心设计如下：</p>
<h3>对抗性构造</h3>
<p>使用 GPT-5 生成包含多对象复杂空间关系的初始提示（clean prompt），再由 GPT-5 对其进行空间扰动（如将"左侧"改为"右侧"、交换两物体相对位置），保持其余空间关系不变，得到 perturbed prompt。用<strong>同一</strong>图像生成模型分别生成两条提示对应的图像，消除模型美学差异的干扰。原始提示生成的图像为 perfect image，扰动提示生成的图像为 perturbed image。</p>
<h3>数据多样性与质量保证</h3>
<p>采用三个 SOTA 生成模型（Qwen-Image、HunyuanImage-2.1、Seedream 4.0），每对随机选用其一。所有偏好对经<strong>人工标注员逐一审核</strong>：（1）验证 perfect image 是否忠实满足所有空间约束；（2）验证 perturbed image 是否确实展现了预期的空间偏差。不满足条件的对直接丢弃。</p>
<h3>数据特点</h3>
<p>与 GenEval 相比，SpatialReward-Dataset 的提示<strong>显著更长</strong>（涉及多对象复杂场景），包含<strong>多重空间关系</strong>，具有更高的空间复杂度和组合多样性。</p>
<h2>SpatialScore 奖励模型</h2>
<h3>模型架构</h3>
<p>以 <strong>Qwen2.5-VL-7B</strong> 作为骨干 H_ 提取图像和文本特征，将原始语言建模头替换为线性奖励头 R_（MLP），预测奖励值： s = R_(H_(c, y)) 其中 c 为指令提示，y 为生成图像。</p>
<h3>奖励训练</h3>
<p>使用 <strong>LoRA</strong> 微调以保留预训练知识。受 HPSv3 启发，将奖励分数建模为<strong>高斯分布</strong> s N(, ^2)：在提示末尾插入特殊 token <code><reward></code>，其最终层嵌入经 MLP 映射为 和 。训练时对每个三元组 (c, y_w, y_l) 进行两次前向传播，基于 <strong>Bradley-Terry</strong> 模型优化： P(y_w y_l c) = (R_(H_(y_w, c)) - R_(H_(y_l, c))) L_Reward() = E_c, y_w, y_l[- P(y_w y_l c)] 训练稳定性增强：从高斯分布采样 1000 次取平均计算损失。训练在 8 张 H20 GPU 上一天完成，学习率 2\!\!10^-6，梯度累积批大小 32。</p>
<h3>奖励模型评估</h3>
<p>构建 365 对高质量偏好对的评估基准，涵盖 1 次扰动和 2--3 次扰动子集。</p>
<p><strong>关键发现：</strong>仅 7B 参数的 SpatialScore 达到 95.8\</p>
<h2>基于 SpatialScore 的在线 RL</h2>
<h3>基础模型与 SDE 采样</h3>
<p>选择 <strong>FLUX.1-dev</strong> 为基础模型（支持长文本，未经后训练）。遵循 Flow-GRPO，将确定性 ODE 转化为等价 SDE 以实现策略探索： x_t+ t = x_t + [v_(x_t, t) + _t^22t(x_t + (1-t)v_(x_t, t))] t + _t t\,</p>
<h3>GRPO 优化与优势计算</h3>
<p>对每个提示 c，策略 _ 通过 SDE 采样生成 G=24 张图像，SpatialScore 为每张评分 R(x_0^i, c)，组内归一化计算优势： A^i = R(x_0^i, c) - mean(\R(x_0^i, c)\)std(\R(x_0^i, c)\)</p>
<h3>Top-k 过滤策略</h3>
<p>作者发现随训练推进，不同难度提示导致<strong>优势偏差</strong>：简单提示产生大量高分样本，组均值过高使部分优质样本获得负优势，优化方向偏离。为此提出 top-k 过滤——按分数排序后仅选取 top-k 和 bottom-k 样本组成子集 S，用于计算均值/标准差及策略更新。GRPO 目标函数为带 clipping 的策略梯度加 KL 散度正则化项。</p>
<p>实验表明 k\!=\!6（从 G\!=\!24 中选 12 个）在采样平衡与多样性间取得最佳权衡，且将训练阶段每提示 NFE 从 24\!\!6\!=\!144 降至 12\!\!6\!=\!72。</p>
<h3>训练细节</h3>
<p>LoRA rank 32，学习率 3\!\!10^-4，importance clipping 范围 1\!\!10^-4，KL 惩罚系数 0.01，32 张 H20 GPU 训练。</p>
<h2>实验结果</h2>
<h3>多基准空间理解评估</h3>
<p>域内 SpatialScore 从 2.18 提升至 <strong>7.81</strong>（3.6）。在域外基准上，DPG-Bench 关系子维度从 0.871 升至 0.932；TIIF-Bench 长文本 BR/AR/RR 分别提升至 0.845/0.715/0.675；UniGenBench^++ 长文本 Layout-2D/3D 提升至 0.891/0.801。所有短文本和长文本设置均获<strong>一致提升</strong>。</p>
<p>对比之下，基于 GenEval 规则奖励训练的 Flow-GRPO 在短文本上有部分提升，但在长文本设置上<strong>严重退化</strong>（TIIF-long BR 从 0.758 降至 0.577，UniGenBench-long 2D 从 0.819 降至 0.445），说明规则基奖励无法泛化至复杂多对象空间关系。</p>
<h3>DPG-Bench 全维度评估</h3>
<p>RL 增强模型在 DPG-Bench 全部五个维度上均超越 Flux.1-dev，整体性能<strong>接近专有 GPT-Image-1</strong>（85.03 vs 85.15），增益不仅限于空间子维度。</p>
<h3>Qwen-Image 上的验证</h3>
<p>在更强的 Qwen-Image 基础模型上，SpatialScore 指导的 RL 同样有效：域内分数从 6.74 提升至 8.25，各文图对齐基准的空间子维度在短/长文本设置上均获一致提升。</p>
<h3>GenEval 基准零样本评估</h3>
<p>尽管训练未使用 GenEval，模型在该基准上仍取得显著零样本提升：Position 从 0.19 升至 0.37，Color Attribute 从 0.45 升至 0.66，Overall 从 0.65 升至 0.78。</p>
<h2>消融实验</h2>
<h3>奖励模型骨干大小</h3>
<p>SpatialScore 在 Qwen2.5-VL-3B/7B/32B 骨干上的配对准确率分别为 89.1\</p>
<h3>Top-k 过滤</h3>
<p>（1）不使用过滤（G\!=\!24 全部更新）：SpatialScore 7.73，NFE 24\!\!6；（2）k\!=\!4：早期收敛快但后期因多样性不足而减速，SpatialScore 7.71，NFE 8\!\!6；（3）k\!=\!6（默认）：SpatialScore 7.81，DPG-Rel 0.932，NFE 12\!\!6，在性能和效率间取得最优。训练曲线显示 top-k 过滤<strong>加速收敛</strong>且最终性能更优或持平。</p>
<h2>定性分析</h2>
<p>定性对比显示，SpatialScore 指导的 RL 模型在复杂多对象空间关系（如"笔记本左侧、咖啡杯在其右前方、盆栽在桌面右缘中央、书架在桌后墙面"等组合约束）上生成更忠实的图像。而 GenEval 引导的 Flow-GRPO 模型不仅空间泛化能力有限，还会<strong>丢失基础模型的长文本跟随能力</strong>（遗漏关键物体如蜡烛、帐篷等），且因依赖目标检测器的规则基奖励，常生成视觉不合理的伪影（如悬浮的地图或夹克）。</p>
<h2>讨论与局限</h2>
<p><strong>核心贡献：</strong>（1）首个面向空间理解的大规模对抗性偏好数据集（80K对）；（2）首个专用空间奖励模型，7B 参数即超越专有模型；（3）top-k 过滤策略缓解 GRPO 中的优势偏差并提升训练效率。</p>
<p><strong>局限与未来方向：</strong></p>
<ul>
<li>当前仅在图像层面验证，未扩展到<strong>视频生成</strong>——视频需同时理解静态空间关系和时间动态变化（如物体移动、位置交换）；</li>
<li>对 sim-to-real 具身仿真场景尤为重要，需生成时间一致且空间准确的视频序列以弥合仿真与现实的差距；</li>
<li>训练数据依赖 SOTA 生成模型的文图对齐能力，数据质量受限于生成模型本身；</li>
<li>奖励模型专注于空间维度，与美学、风格等其他维度的联合优化有待探索。</li>
